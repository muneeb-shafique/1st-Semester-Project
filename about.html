About the Project

This project aims to manage and organize student data, including their grades, efficiently. The system is built using Python and helps teachers or administrators track student performance.

Features

  1. User Input

    The user (teacher/admin) will input the following details for each student:

    • Student Name
    • Roll Number
    • Marks for each subject (e.g., AICT, PF, Calculus, etc.)
    • Other necessary details (optional, like class, section, etc.)

  2. Calculations

    • Grades: Calculates grades for each subject and the overall grade.
    • Percentage: Calculates the percentage based on total marks and maximum possible marks.
    • Marks: Calculates individual subject marks and total marks.
    • Position: Ranks students based on their total marks (1st, 2nd, 3rd, etc.).

  3. Report Generation

    Generates a Report Card for each student in the form of an HTML file. The report card includes:

    • Student’s name, roll number, and marks for each subject.
    • Obtained marks, total marks, percentage, grade, and position.
    • The report is styled with HTML for better presentation.

  4. Additional Features

    • Data Validation: Ensures entered marks are numeric and within the valid range.
    • Report Accessibility: The generated report card is easily shareable and printable.
Flow of the System
  • User Input: Teacher/admin enters student data via a user interface.
  • Processing: The system calculates marks, percentage, grades, and ranks.
  • Report Generation: After processing, the system generates the student’s report card in HTML format.
  • Output: The generated report card can be viewed, edited, or printed by the user.
